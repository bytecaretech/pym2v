# Contributing

1. Install [uv][1]
1. Install [task][2]
1. Setup the project: `task setup`
1. Create a dedicated development branch: `git checkout -b <hyphen-delimited-branch-name>`
1. Add your changes
1. Run linters: `task lint`
1. Run tests: `task test`
1. Commit changes and open a pull request


[1]: https://docs.astral.sh/uv/getting-started/installation/
[2]: https://taskfile.dev/docs/installation
